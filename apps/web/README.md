# @pearlhub/web

Next.js 16 App Router frontend for PearlHub 2.0.

## Commands

- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test:e2e`

## Notes

- Theme tokens are in `src/styles/globals.css`
- AI concierge route is in `src/app/api/concierge/route.ts`
- Marketplace pages are in `src/app/(marketplace)`
